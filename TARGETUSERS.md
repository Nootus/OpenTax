# 🎯 Target Users of OpenTax

OpenTax is designed as a national-scale, open, community-driven tax computation engine for India.  
Its architecture, licensing, and API-first design make it useful for a wide range of users — from individual taxpayers to enterprise-grade financial platforms.

This document outlines **who benefits from OpenTax** and **how they can use it effectively**.

---

# 🧑‍💻 1. Developers & Engineering Teams

### Who they are:
- Fintech developers  
- Payroll system engineers  
- Accounting software teams  
- Startup founders building tax-enabled products  
- API integrators  
- Open-source contributors  

### How they benefit:
- Use OpenTax as a **plug-and-play tax engine**  
- Avoid re-implementing complex tax rules  
- Build tax-enabled features faster (ITR filing, payroll, capital gains, etc.)  
- Rely on a **deterministic, audited, community-maintained** tax logic  
- Integrate via REST API or embed the engine directly  
- Reduce compliance risk by using a shared, validated standard  

### Typical use cases:
- Payroll tax computation  
- Salary slip generation  
- Investment apps calculating tax impact  
- Accounting platforms generating ITR JSON  
- Fintech apps offering tax previews or tax planning  
- SaaS platforms embedding tax logic without maintaining it internally

OpenTax is built to serve:

- Developers
